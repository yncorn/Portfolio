Data Collection and Storage
  • API Source: TMDB API for movie details, descriptions, genres, and user reviews.
    • Number of User Rows: 1,999
    • Number of Movie Review Rows: 44,878
    • Number of Movie Rows: 547
  • Collected data stored in Azure Blob Storage.
  • Then, Azure Data Factory automates the data collection and storage process.
Data Preparation and Model Training
Data Preprocessing

Use Azure Databricks to clean, transform, and integrate data.

Preprocessed NoSQL data to SQL data for training.

Model Training

Utilize Azure Machine Learning for model training, then store the trained model in Azure Cosmos DB.

Algorithms: ALS (Alternating Least Squares) and SVD (Surprise Singular Value Decomposition).

Model Deployment

Deployed on Azure Container Instances managed by Azure Kubernetes Service.

Developed using Power Apps

Allows users to input movie ratings and receive recommendations from the deployed model.
